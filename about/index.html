about

datastruct-visual is a personal project built to study frontend development independently — no frameworks, no dependencies, just HTML, CSS and JavaScript.

the idea came from a simple need: understand data structures by seeing them, not just reading about them. each visualization was built alongside its real implementation, so the animation and the code tell the same story.

if you're struggling with stacks, lists, trees or graphs — this is for you too. the goal was always to make these concepts as visual and intuitive as possible.

— Ana Clara Martins
